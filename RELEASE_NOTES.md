## 🚀 **Installation Guide**

### 📥 **Download APK**
1. Navigate ke folder: `build/app/outputs/flutter-apk/`
2. Download file: `app-release.apk` (23.0MB)

### 📱 **Install on Android**
1. Enable "Install from Unknown Sources" di Android settings
2. Open APK file
3. Tap "Install"
4. Wait for installation to complete
5. Open app
